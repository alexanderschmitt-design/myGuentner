How Feature Flags Work

Feature flags are stored in localStorage and take effect immediately on all pages. Each flag is stored as gpc_feature_* with a value of "1" (enabled) or "0" (disabled). The homepage accordion toggles above share a single JSON entry mygpc_home_accordions. Other pages read these values on load to show or hide functionality. No server restart is required.
